height 속성

(기본 값 : 0)

-- 속성 값--

auto 브라우저에 의해 자동으로 계산하여 지정
(기본적으로 컨텐츠 영역의 내용만큼 높이를 갖는다.)
length 고정값으로 지정 (ex. px, em, ...)
percent 부모 요소의 height에 상대적인 크기를 지정
(단, 부모 요소가 명시적으로 height 값이 있어야 한다.)


height는 content 영역의 높이를 제어할 때 사용하는데 이때 auto가 아닌 특정한 값을 지정하여 사용하면
width 속성과 마찬가지로 box model 영역에서 margin 영역을 제외한 모든 영역에 대해 영향을 받는다.



예를 들어,
width: 100px;
height: 100px;
padding: 10px;
border: 15px solid black;

이와 같이 선언된다면 요소의 총 세로 크기는 150px가 된다. height 또한 padding, border 영역에 대해 추가로 영향을 받기 때문



100px content + (10px * 2) padding + (15px * 2) border = 150px 이기 때문.

또한 height는 %를 이용하여 크기를 지정할 수 있다.

.parent {
width: 300px;
height: 200px;
border: 20px solid red;
}

.child {
height: 50%;
background: red;
}
