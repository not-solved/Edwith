width 속성

(기본 값 : 0)

-- 속성 값--

auto 브라우저에 의해 자동으로 계산하여 지정
(요소의 레벨 기본 특성에 따라 다르게 동작)
length 고정값으로 지정 (ex. px, em, ...)
percent 부모 요소의 width에 상대적인 크기를 지정


width는 content 영역의 너비를 제어할 때 사용하는데 이때 auto가 아닌 특정한 값을 지정하여 사용하면
그 크기는 box model 영역에서 margin 영역을 제외한 모든 영역에 대해 영향을 받는다. (content, padding, border)



예를 들어,
width: 100px;
padding: 20px;
border: 10px solid black;

이와 같이 선언된다면 요소의 총 가로 크기는 160px가 된다. width는 padding, border 영역에 대해 영향을 받기 때문



100px content + (20px * 2) padding + (10px * 2) border = 160px 이기 때문.

또한 width는 %를 이용하여 크기를 지정할 수 있다.

.parent {
width: 300px;
border: 20px solid red;
}

.child {
width: 50%;
padding: 20px;
border: 10px solid black;
}
