@media(at media)

CSS2 Media Type을 확장했기 때문에 선언 방식은 동일

@media mediaqueries { /* style rules */ }


@media로 시작하며 이는 '이제부터 미디어 쿼리를 시작한'라는 뜻이다.
뒤에 미디어 쿼리 구문 (mediaqueries)이 따르며 이어서 중괄호 ( {} )를 이용해 스타일 규칙이 들어간다.
미디어 쿼리 구문은 논리적으로 평가되며 참이면 스타일 규칙이 적용되고 거짓이면 무시된다.
미디어 쿼리 구문은 미디어 타입과 미디어 특성으로 이루어져있다.

미디어 타입

- all, braille, embossed, handheld, print, projection, screen, speech, tty, tv


자주 쓰이는 것은 all, print, screen 정도이며, 그 중 screen이 거의 대부분이다.
화면을 출력하는 디스플레이가 있는 미디어들은 전부 screen에 속하기 때문에 현실적으로 고려해야 하는 미디어들은 전부 이것에 해당.
print 또한 간혹 사용된다.
all 타입은 모든 미디어에 적용되는 타입. 미디어를 구분하는 용도가 아니기 때문에 유용하게 사용되지는 않는다.


미디어 특성

- width, height, device-width, device-height orientation, aspect-ratio, device-aspect-ratio, color, color-index, monochrome, resolution, scan, grid


자주 쓰이는 것은 width, orientation 정도
width : 뷰포트의 너비 (브라우저 창의 너비)를 의미 (스크린의 크기가 아님)
orientation : 미디어가 세로모드인지 가로모드인지 구분
- 미디어 쿼리에서는 width와 height를 비교하여 height가 width 이상이면 세로모드, 반대인 경우는 가로모드로 해석한다.
- 세로모드에서는 portrait, 가로모드에서는 landscape 키워드와 매칭된다.
